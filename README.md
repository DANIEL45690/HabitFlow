# 🌊 HabitFlow

**HabitFlow** — это Telegram Mini App для трекинга привычек и задач с подсчётом непрерывных дней (streak). Сохраняйте данные на устройстве, отмечайте прогресс и формируйте полезные привычки каждый день.

## ✨ Возможности

| Функция | Описание |
|---------|----------|
| ✅ **Управление задачами** | Добавляй, отмечай выполняй и удаляй задачи |
| 🔥 **Трекер привычек** | Подсчёт дней подряд (streak) для каждой привычки |
| 📊 **Статистика** | Количество задач, выполненных задач и привычек |
| 💾 **Локальное сохранение** | Все данные хранятся в `localStorage` вашего устройства |
| 🎨 **Тёмная тема** | Приятный глазу современный дизайн |
| 📱 **Адаптивность** | Работает на любых устройствах с Telegram |

## 🛠 Технологии

- **Backend**: Python 3.8+, `python-telegram-bot` v20+
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Сервер**: Встроенный HTTP-сервер (для WebApp)
- **API**: Telegram Bot API
